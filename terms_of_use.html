Условия использования

Приложение "Social Balance": Означает мобильное приложение, созданное для обеспечения контроля родителей над активностью и обучением их детей.

Права и обязанности

Пользователи:

  • Пользователи обязуются использовать приложение с уважением к правам других.
  • Родители имеют право и обязанность использовать функции удаленного управления для надлежащего воспитания и контроля своих детей.
  • Дети обязуются использовать приложение в соответствии с установленными правилами и не нарушать инструкции родителей.

Конфиденциальность и безопасность

Приложение прикладывает максимальные усилия для обеспечения безопасности и конфиденциальности данных пользователей.

Личная информация хранится в защищенной среде, и доступ к ней предоставляется только авторизованным лицам.

Использование данных

Сбор данных:

  • Приложение собирает минимальное количество данных, необходимых для своего функционирования.
  • Геолокационные данные используются только для предоставления функции отслеживания местоположения детей.

Использование данных:

Собранные данные используются для улучшения функциональности приложения и персонализации пользовательского опыта.

Ограничения использования

  • Запрещено использование приложения в незаконных целях или способах, не предусмотренных разработчиками.
  • Родители несут ответственность за настройку функций контроля в соответствии с рекомендациями разработчиков.

Ответственность

Разработчики не несут ответственности за неправильное или ненадлежащее использование приложения со стороны пользователей.
